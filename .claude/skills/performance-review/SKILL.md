# Performance Review System

## When to Use This Skill

Use this skill when you need to:
- Design a structured performance review process (annual, quarterly, 360°)
- Create evaluation forms and rating scales
- Link performance scores to KPIs and business goals
- Build development plans from review outcomes

---

## Core Principle

MEASURE WHAT MATTERS — A REVIEW THAT TAKES 3 HOURS BUT CHANGES NOTHING IS WORSE THAN NO REVIEW AT ALL. FOCUS ON BEHAVIOR, OUTCOMES, AND GROWTH.

---

## Phase 1: Brief

| Input | What to Ask | Default |
|-------|------------|---------|
| **Review frequency** | Annual, semi-annual, or quarterly? | Semi-annual |
| **Review type** | Self-assessment, manager review, or 360°? | Manager + self |
| **Rating scale** | 1-5 numeric, descriptive levels, or OKR-style? | 1-5 with descriptors |
| **Linkage** | Tied to compensation, promotion, or development only? | Development + compensation |

---

## Phase 2: Evaluation Framework

### Core Competency Areas
1. **Job Performance** — quality and quantity of work vs KPIs
2. **Collaboration** — teamwork, communication, contribution to team goals
3. **Initiative** — problem-solving, proactiveness, continuous improvement
4. **Leadership** — (for managers) team development, decision-making
5. **Alignment** — living company values and mission

### Rating Scale (1–5)
| Score | Label | Description |
|-------|-------|-------------|
| 5 | Exceptional | Consistently exceeds all expectations |
| 4 | Strong | Meets and often exceeds expectations |
| 3 | Meets Expectations | Reliably delivers on core responsibilities |
| 2 | Needs Improvement | Partially meets expectations with gaps |
| 1 | Unsatisfactory | Fails to meet core expectations |

---

## Anti-Patterns
- Rating everyone 3/5 to avoid conflict
- No specific examples or evidence for ratings
- Review disconnected from daily feedback
- No follow-up after the review meeting
