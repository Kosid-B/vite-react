# Talent Acquisition Strategy

## When to Use This Skill

Use this skill when you need to:
- Build a sourcing strategy for a specific role
- Design a structured interview and scoring process
- Reduce time-to-hire and improve quality-of-hire metrics
- Create candidate experience from application to offer

---

## Core Principle

HIRE FOR POTENTIAL AND VALUES FIRST, SKILLS SECOND — SKILLS CAN BE TAUGHT, CHARACTER AND DRIVE CANNOT.

---

## Phase 1: Role Brief

| Input | What to Ask | Default |
|-------|------------|---------|
| **Role** | Job title, team, and reporting line? | As per org chart |
| **Timeline** | When do you need someone in the seat? | 60 days |
| **Must-have skills** | Top 3 non-negotiable requirements? | From JD |
| **Nice-to-have** | Bonus skills that accelerate success? | From JD |
| **Culture fit** | Key values and working style for this team? | Company values |

## Phase 3: Interview Process

```
## Interview Pipeline — [Role]

### Stage 1: Recruiter Screen (30 min)
- Verify must-have requirements
- Assess motivation and salary expectation
- Scorecard: Pass / No Pass

### Stage 2: Hiring Manager Interview (60 min)
- Technical/functional competency (role-specific questions)
- Problem-solving scenario
- Scorecard: 1–5 per competency area

### Stage 3: Team Interview (45 min)
- Collaboration and culture fit
- Day-in-the-life scenario
- Scorecard: 1–5 · Must hit 4+ average

### Stage 4: Case Study / Work Sample (for senior roles)
- Real problem relevant to the role
- Scored on approach, analysis, and communication

### Offer Stage
- Reference check (2 professional references)
- Offer letter with 30/60/90-day success criteria
```

---

## Anti-Patterns
- Unstructured interviews where interviewers ask different questions
- Hiring based on gut feel without a scorecard
- No candidate feedback loop (hurts employer brand)
- Slow process — losing top candidates to faster competitors
