# หาลูกค้าจริงราย แรก — Outreach Script (ฐานลูกค้าเดิม)

> **เป้าหมายเดียว:** ได้ **signup จริง 5 ราย** (ไม่ใช่แค่ traffic) ภายใน 2 สัปดาห์
> **ทำไมต้อง manual:** GA4 มี 175 active/สัปดาห์ แต่ Supabase = 1 user (แอดมินเอง) → organic ยังไม่ปิดการขาย

---

## หลักการ (อย่าหว่าน — เจาะ)
1. เริ่มจาก **คนที่เคยจ่ายเงินให้เราแล้ว** (เชื่อใจอยู่แล้ว) ไม่ใช่คนแปลกหน้า
2. **ไม่ขายของ — ชวนลอง + จับมือทำให้ (concierge)** ให้เขาได้ "aha moment" ก่อน
3. ทุกข้อความจบด้วย **1 action ชัดเจน** (กดลิงก์ / นัดเวลา 15 นาที)
4. **แนบลิงก์ที่ลองได้ทันทีไม่ต้องสมัคร** (guest mode ที่เพิ่งทำ) — ลด friction

## 🤝 Concierge onboarding (หัวใจ — ทำให้เขาสำเร็จ)
เมื่อมีคนตอบรับ **อย่าปล่อยให้ลองเอง** — นัด 15 นาที แล้ว:
1. พาสร้างโครงการ + เลือกมาตรฐาน (มอก./ISO ของเขา)
2. รัน **Gap Analysis** ให้ดู → เห็น % ความพร้อม + ช่องโหว่จริง (= aha moment)
3. ตอนจบ: "ระบบเก็บงานให้ ถ้าอยากบันทึกถาวร สมัครฟรีตรงนี้ครับ" → **ได้ signup จริง**
