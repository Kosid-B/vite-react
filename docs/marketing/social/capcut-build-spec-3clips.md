# CapCut Build Spec — 3 คลิป ISO (ป้อน CapCut สร้างได้เกือบอัตโนมัติ)

> ⚠️ **ข้อจริง:** CapCut ไม่มี API ให้สั่งจากภายนอกแบบ 100% อัตโนมัติ — แต่ใช้ 3 ฟีเจอร์ในตัวช่วยได้เกือบหมด:
> **① Text to Video** (วางสคริปต์ → สร้างคลิปพร้อม footage+เสียง+ซับ) · **② Text to Speech** (พากย์อัตโนมัติ) · **③ Auto Captions** (ซับอัตโนมัติ)
> ไฟล์นี้จัดสคริปต์ให้ "วางแล้วสร้างได้เลย" + ระบุตั้งค่า + จุดที่ต้องสลับเป็นภาพหน้าจอแอปเอง

---

## 🅰️ วิธีเร็วสุด (auto 80%) — ใช้ CapCut "Text to Video"
**ขั้นตอน:** เปิด CapCut → **สร้างใหม่ → ข้อความเป็นวิดีโอ (Text to Video)** → วาง "บทวางในช่อง Script" ด้านล่าง → ตั้งค่า → Generate → ได้ draft พร้อมซับ+เสียง → **สลับฉากที่ต้องโชว์แอปด้วยคลิปหน้าจอจริง** → Export

**ตั้งค่าทุกคลิป:**
- อัตราส่วน: **9:16 (แนวตั้ง)**
- เสียงพากย์ (TTS): ภาษาไทย เสียงผู้ชาย โทน "มั่นใจ/เป็นกันเอง" (เช่น "ธีร์/ก้อง" ถ้ามี)
- ซับไตเติล: **เปิด · ตัวหนา · พื้นหลังทึบ** · สีเน้นคำสำคัญ = ส้ม #f59e0b
- เพลง: จังหวะกลาง ๆ ไม่มีเนื้อร้อง (หมวด Corporate/Upbeat) เบา ๆ ~15%

## 📋 คลิป 1 — "7 เอกสารที่ Auditor ขอ"
### 🎙️ บทวางในช่อง Script (คัดลอกวางทั้งก้อน)
```
ถ้าพรุ่งนี้ auditor เดินเข้าโรงงานคุณ คุณหยิบเอกสาร 7 อย่างนี้ได้ใน 5 นาทีไหม?
เอาจริง ๆ ส่วนใหญ่ตกที่ข้อ 3 กับข้อ 6 นี่แหละ
หนึ่ง นโยบายคุณภาพที่วัดผลได้. สอง ทะเบียนคุมเอกสาร. สาม บันทึกการตรวจภายใน.
สี่ ประชุมทบทวนฝ่ายบริหาร. ห้า การแก้ไขข้อบกพร่อง. หก บันทึกการอบรมพนักงาน. เจ็ด การประเมินผู้ขาย.
โรงงานส่วนใหญ่ทำครบ แค่หาหลักฐานไม่เจอตอน audit เลยได้ NC น่าเสียดายมาก
อยากรู้ว่าโรงงานคุณพร้อมกี่ % แล้วยังขาดอะไร เช็กฟรีที่ลิงก์ในไบโอ พิมพ์คำว่า ISO ในคอมเมนต์ เดี๋ยวส่งวิธีให้
```
### 🎬 ตารางประกอบ (สลับ media ตามนี้หลัง Generate)
| ช่วง | ข้อความบนจอ | media ที่ควรใช้ | keyword ค้น footage (ถ้าใช้ stock) |
|---|---|---|---|
| Hook | **หยิบครบไหม?** | มือเปิดแฟ้ม/ลิ้นชักหาเอกสาร | "searching documents drawer" |
| ลิสต์ 1–7 | ตัวเลข 1–7 เด้ง | **สลับเป็นหน้าจอแอป ISO Advisor** ไล่เช็กทีละข้อ + แถบ % วิ่ง | (ใช้ screen record จริง) |
| ปิด | **เช็กฟรี · ลิงก์ไบโอ** | การ์ด /start + ปุ่มส้ม | (ทำ text layer เอง) |

## 📋 คลิป 2 — "จ้างที่ปรึกษา ISO ราคาเท่าไร?"
### 🎙️ บทวางในช่อง Script
```
ค่าจ้างที่ปรึกษา ISO อยู่ที่ห้าหมื่น ถึงสองแสนกว่าบาท
แต่เดี๋ยวก่อนครับ อย่าเพิ่งจ่าย
หลายคนจ่ายแพงเกินจำเป็น เพราะไม่รู้ว่าตัวเองขาดแค่ไหน จริง ๆ อาจขาดแค่ไม่กี่จุด
ลองประเมินความพร้อมฟรีก่อน แล้วค่อยตัดสินใจว่าจะทำเองส่วนไหน จ้างที่ปรึกษาช่วยเฉพาะจุดที่ยากจริง
ประเมินฟรีที่ลิงก์ในไบโอ หรืออยากให้ที่ปรึกษา 20 ปี ดูให้ โทรมาได้เลย
```
### 🎬 ตารางประกอบ
| ช่วง | ข้อความบนจอ | media | keyword ค้น footage |
|---|---|---|---|
| Hook | **฿50,000–250,000** | ตัวเลขราคาเด้งใหญ่ | "money calculator invoice" |
| หยุด | **อย่าเพิ่งจ่าย** | มือทำท่าหยุด + ราคาโดนกากบาท | "stop hand gesture" |
| เนื้อหา | พร้อม 78% · ขาด 4 เอกสาร | **หน้าจอแอปประเมิน** | (screen record จริง) |
| ปิด | **ประเมินฟรี · 081-781-7773** | การ์ด /start + เบอร์โทร | (text layer เอง) |

## 📋 คลิป 3 — "PDPA ไม่ใช่แค่ประกาศหน้าเว็บ"
### 🎙️ บทวางในช่อง Script
```
คิดว่ามีประกาศ PDPA หน้าเว็บแล้ว ปลอดภัยเหรอครับ? ยังไม่พอครับ
PDPA ต้องมีหลังบ้านด้วย
บันทึกกิจกรรมการประมวลผลข้อมูล หลักฐานความยินยอม ขั้นตอนรับคำขอจากลูกค้า
สัญญากับคู่ค้า แผนรับมือข้อมูลรั่ว และบันทึกการอบรมพนักงาน
เช็กฟรีว่าธุรกิจคุณขาดเอกสาร PDPA อะไร ที่ลิงก์ในไบโอ พิมพ์คำว่า PDPA ในคอมเมนต์
```

| พลิก | **❌ ยังไม่พอ** | ถูกกลายเป็นกากบาทแดง | "red cross error" |
| เนื้อหา | ROPA·Consent·DSAR·DPA·Breach·อบรม | **หน้าจอแอปเช็ก PDPA** | (screen record จริง) |

## 🅱️ ทางเลือก: ให้ CapCut พากย์+ซับ แต่ใช้ภาพหน้าจอแอปเราเอง (คุณภาพสูงกว่า)
1. อัด **หน้าจอแอปจริง** ทีละสเต็ป (ISO Advisor / ประเมิน %) แนวตั้ง
2. CapCut → นำเข้าคลิปหน้าจอ → เรียงตามตาราง
3. **Text to Speech:** วางบทพากย์ (ก้อนข้างบน) → เลือกเสียงไทย → วางทับ timeline
4. **Auto Captions:** กด "คำบรรยายอัตโนมัติ" → CapCut ถอดเสียงเป็นซับให้ → จัดสไตล์ตัวหนา
5. เพิ่ม text layer ตัวเลข/คำเน้น (ส้ม) ตอน hook + ปิดท้าย
6. Export 1080×1920 · 30fps
