# ISO Short — สไตล์อินโฟกราฟิก (เฟรม + วิธีทำ motion ใน CapCut)

> อ้างอิงสไตล์: คลิป @หุ้นวันละนิด "10 หุ้นหุ่นยนต์ 2026" (หัวเรื่อง + ภาพ + 3 คอลัมน์)
> ไฟล์เฟรม: `iso-short-infographic-frame.png` (1080×1920) · source แก้ได้: `iso-short-infographic-frame.html`
> จุดเด่น: ไม่ต้องถ่ายหน้า/หน้าจอ — ใช้เฟรมนิ่ง + motion + เสียงพากย์

## 🎬 ทำเฟรมนิ่ง → คลิปมีชีวิต ใน CapCut (~5 นาที)
1. **นำเข้า** `iso-short-infographic-frame.png` เป็นพื้นหลังเต็มจอ (9:16)
2. **Ken Burns / ซูมช้า:** เลือกภาพ → เพิ่ม keyframe scale 100% → 108% ตลอดคลิป (ภาพนิ่งดูมีชีวิต)
3. **เผยทีละส่วน (ให้ตรงจังหวะเสียง):** ตัดภาพเป็นเลเยอร์ หรือใช้ "Overlay" ครอบส่วนล่างด้วยสี่เหลี่ยมทึบ แล้ว keyframe เลื่อนหลบทีละช่วง:
   - 0–3s เห็นแค่หัวเรื่อง (จี้จุด)
   - 3–8s เผยการ์ด "78% พร้อม" + แถบวิ่ง (ใส่ sticker ↑ หรือ animate แถบ)
   - 8–14s เผย 3 คอลัมน์ทีละอัน (effect "ป็อป")
   - 14–18s เน้นปุ่ม "เริ่มฟรี" (zoom + สั่นเบา ๆ)
4. **Text to Speech:** วางบทพากย์ด้านล่าง → เลือกเสียงไทยผู้ชาย โทนมั่นใจ
5. **เพลง:** Corporate/Tech เบา ๆ ~15% · **ซับ:** Auto Captions ตัวหนา
6. Export 1080×1920 · 30fps

## 🔁 ทำเป็น "ชุด" (series) แบบช่องอ้างอิง
เฟรมนี้ = แม่แบบ · เปลี่ยนหัวเรื่อง + ตัวเลข + 3 คอลัมน์ ทำได้อีกหลายตอน:
- ตอน PDPA: "ธุรกิจไทยเสี่ยงโดนปรับ PDPA — เช็กฟรี"
- ตอน มอก.: "ขอ มอก. ต้องทำอะไรบ้าง"
- ตอนที่ปรึกษา: "จ้างที่ปรึกษา ISO ราคาเท่าไร"
