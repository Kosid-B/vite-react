# Concierge Seed — Phase 1 Execution Kit (Seed Supply)

> ชุดเครื่องมือให้ทีม B. Training **ลงมือเฟส 1 ได้ทันที** (จาก `CONCIERGE-SEED-CHECKLIST.md`):
> เชิญ **ที่ปรึกษา/ผู้ให้บริการ 10–15 ราย** ในเครือข่ายมาเปิดหน้าร้านจริงบน `/b` เพื่อให้ตลาดไม่ว่าง
> **เป้าเฟส 1 (สัปดาห์ 1–2): ≥10 ร้าน published จริง**
>
> ⚠️ กติกา: เชิญ **ธุรกิจจริง** เท่านั้น — ห้ามสร้างร้านปลอม/รีวิวปลอม (สอดคล้อง AggregateRating ซื่อสัตย์)
> การส่งข้อความหาคนจริงเป็นงานของทีม (เอกสารนี้เตรียม "สารตั้งต้น" ให้พร้อมส่ง)

## A. รายชื่อเป้าหมาย (ใครควรเชิญก่อน)
เรียงตามโอกาสปิดง่าย→ยาก (เริ่มจากคนที่ไว้ใจเราอยู่แล้ว):
1. **ลูกค้าเก่า/ที่ปรึกษาในเครือข่าย B. Training** (ISO, อบรม, ระบบคุณภาพ) — เชื่อใจเราแล้ว ปิดง่ายสุด
2. **พันธมิตร referral** ที่เคยส่งงานให้กัน (บัญชี, กฎหมาย, การตลาด, ออกแบบ)
3. **วิทยากร/ผู้เชี่ยวชาญ** ที่เคยร่วมงานอบรม
4. **โรงงาน/ผู้รับจ้างผลิตรายเล็ก** ที่รู้จัก (จ้างผลิต/OEM ย่อย)

> เลือกให้ **กระจายหมวด DBD** อย่างน้อย 4–5 หมวด เพื่อให้สารบัญ `/b` ดูมีชีวิต ไม่กระจุกหมวดเดียว

## C. คู่มือ 1 หน้า ส่งให้ผู้ถูกเชิญ (เปิดร้านเอง ~5 นาที)
> ส่งลิงก์ `https://ceoaithailand.org/start` แล้วแนบขั้นตอนนี้:
1. เข้า **ceoaithailand.org** → เริ่มใช้งาน (ฟรี 15 วัน ไม่ต้องใส่บัตร)
2. เมนูซ้าย → **"หน้าร้านของฉัน"**
3. ทำตามการ์ด **"🚀 เปิดร้านพร้อมรับงาน B2B ใน ~5 นาที"** (3 ก้าว):
   - ยืนยัน **หมวดธุรกิจ (DBD)** ให้ตรง
   - กด **"✨ ให้ AI Agent เขียน"** ที่ช่องจุดขาย → ปรับข้อความให้เป็นคุณ
   - กรอกเบอร์/LINE + บริการเด่น 2–3 อย่าง
4. กด **"🚀 เผยแพร่หน้าร้าน"** → คัดลอกลิงก์ร้านไปใช้ได้เลย
5. (แนะนำ) กด **"📨 ดูงานรอเสนอราคา"** เผื่อมีงานตรงหมวดให้เสนอราคาทันที

## D. ตารางติดตาม (คัดลอกไป Google Sheets)
| # | ชื่อธุรกิจ/ผู้ติดต่อ | หมวด DBD | ช่องทางเชิญ | วันที่เชิญ | สถานะ | slug ร้าน | หมายเหตุ |
|---|---|---|---|---|---|---|---|
| 1 | | | LINE/อีเมล/FB | | เชิญแล้ว / เปิดร้านแล้ว / ยังไม่ตอบ | | |
| 2 | | | | | | | |
| … (เติมถึง 15) | | | | | | | |

**นิยามสถานะ:** เชิญแล้ว → ตอบรับ → เปิดร้าน(draft) → **เผยแพร่แล้ว (นับเป็นเป้า)** → มี RFQ/ดีลแรก (ส่งต่อเฟส 2)

## E. เกณฑ์ผ่านเฟส 1 → ไปเฟส 2
- [ ] ร้าน **published จริง ≥ 10** (ธุรกิจจริง ไม่ใช่ทดสอบ)
- [ ] กระจาย **≥ 4 หมวด DBD**
- [ ] ทุกร้านมีจุดขาย (VP) + ช่องทางติดต่อครบ
- [ ] ตรวจว่าทุกร้านขึ้นในสารบัญ **/b** ถูกหมวด
→ ครบแล้วไป **เฟส 2** (จุดชนวน demand + จับคู่ RFQ) ตาม `CONCIERGE-SEED-CHECKLIST.md`

## F. วัดผล (ดูที่ไหน)
- จำนวนร้าน published — สารบัญ **/b** หรือหน้า `ซื้อขาย B2B` (รายการ "ธุรกิจในระบบ")
- GA4 event **`storefront_published`** + **`seller_onboard_step`** (วัดว่าคนเปิดร้านเสร็จจริง + ใช้เวลาเท่าไร)
