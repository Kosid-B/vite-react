# Promotion Engine — เครื่องยนต์การตลาดที่ "วิ่งจริง" รายสัปดาห์

> **ทำไมมีเอกสารนี้:** Brand Review 7P ให้ Promotion แค่ 2.5/5 — คอขวดอันดับ 1 ของทั้งแบรนด์
> ปัญหาไม่ใช่ "ขาดเครื่องมือ" (มีครบ: GA4, LINE, SEO, บทความ, เว็บ, outreach playbook)
> ปัญหาคือ **ไม่มีจังหวะที่วิ่งสม่ำเสมอ + ยิงกระจาย 3 เสียง (ISO / มือใหม่ / แม่ค้า) พร้อมกัน**
> **ทางแก้:** เลือก **1 beachhead segment** + ตั้ง **drumbeat รายสัปดาห์** ที่ทำได้จริง แล้ววัดด้วย UTM

---

## 1) 🎯 การตัดสินใจ: เจาะ segment เดียวก่อน — **โรงงาน/SME ที่ต้องทำ ISO/มอก.**
### เจาะเฉพาะ **ฐานลูกค้า + ผู้เคยอบรมเดิมของ B. Training Consultant** เป็นหัวหาด

**ทำไมเลือกอันนี้ (ไม่ใช่ที่อื่น):**
| เกณฑ์ | ISO/มอก. (เลือก) ✅ | มือใหม่/ตกงาน | แม่ค้าออนไลน์ |
|---|---|---|---|
| ความอุ่น (เข้าถึงได้เลย) | 🟢 มีรายชื่อ/เบอร์/LINE อยู่แล้ว | 🔴 ต้องหาใหม่ | 🔴 ต้องหาใหม่ |
| ความเจ็บชัด (pain) | 🟢 audit ใกล้ = เจ็บจริง จ่ายเพื่อแก้ | 🟡 กว้าง ไม่เร่ง | 🟡 หลากหลาย |
| กำลังจ่าย (WTP) | 🟢 จ่ายที่ปรึกษาอยู่แล้ว | 🔴 งบน้อย | 🟡 น้อย-กลาง |
| **แต้มต่อที่ลอกไม่ได้** | 🟢 **ที่ปรึกษา 20 ปี = ตรงงานเป๊ะ** | 🟡 ทั่วไป | 🔴 ไม่ตรง |
| สินค้าตรงความต้องการ | 🟢 มีเครื่องมือ ISO/มอก. จริง | 🟡 | 🟡 |

> **หลักหัวหาด (beachhead):** เจาะแคบที่ "เจ็บ + เข้าถึงได้ + เรามีแต้มต่อ" ให้ชนะเด็ดขาดก่อน
> แล้วค่อยขยาย — ไม่ใช่หว่านกว้างแล้วบางทุกที่ · **segment อื่นยังไม่ทิ้ง แค่ยังไม่ยิงสัปดาห์นี้**

## 2) 📣 ข้อความเดียวสำหรับหัวหาดนี้ (Message-Market Fit)
ใช้ **3 แรงขับ** (จาก skill `buying-triggers`) เจาะ ISO/มอก. โดยเฉพาะ:

- 🩹 **ขจัดความทุกข์:** "ไม่ต้องปวดหัวเตรียมเอกสาร audit เป็นสัปดาห์ หรือรอที่ปรึกษาว่าง"
- ⚡ **ความสะดวก:** "รู้ทันทีว่าโรงงานพร้อมกี่ % + ขาดเอกสารอะไรก่อน audit"
- ✨ **อารมณ์:** "มั่นใจทุกครั้งที่เจ้าหน้าที่มาตรวจ — เหมือนมีที่ปรึกษา 20 ปีดูหลังให้"

**พาดหัวหลัก (ใช้ทุกช่องทาง):**
> **"เตรียม ISO/มอก. ให้พร้อม audit — ด้วย AI ที่ออกแบบโดยที่ปรึกษาไทย 20 ปี · เริ่มฟรี"**

## 3) 📡 ช่องทาง — ทำอะไร / ยังไม่ทำอะไร (โฟกัส)
| ช่องทาง | สัปดาห์นี้ | เหตุผล |
|---|---|---|
| **Warm outreach** (email/LINE/โทร) | ✅ ทำหลัก | อุ่นสุด ปิดเร็วสุด — TRACK A ใน `FIRST-CUSTOMER-OUTREACH.md` |
| **SEO บทความ ISO/มอก.** | ✅ ทำ | โรงงาน Google หา "ขอ มอก. ต้องทำอะไร" — มีบทความเริ่มแล้ว |
| **LinkedIn (founder)** | ✅ ทำ | B2B + ดัน trust ที่ปรึกษา 20 ปี — `LINKEDIN-STRATEGY.md` |
| **FB group** (เจ้าของธุรกิจ/ISO) | 🟡 เบา ๆ | เข้าไปช่วยตอบ ยังไม่ตั้งกลุ่มเอง — `FACEBOOK-GROUP-PLAN.md` |
| TikTok/Shorts | 🔴 พักไว้ | เหมาะแม่ค้า/Gen Z ไม่ใช่ผู้จัดการโรงงาน — เก็บไว้ตอนขยาย segment |

## 4) 🧱 4 เสาคอนเทนต์ (Content Pillars) — หมุนวนรายสัปดาห์
1. **สอน/ให้ความรู้** (top funnel, SEO): "ขอ มอก. ต้องทำอะไรบ้าง", "เอกสาร ISO 9001 ที่ audit ขอ 7 อย่าง"
2. **เบื้องหลังที่ปรึกษา 20 ปี** (trust, founder): เคสจริง/ข้อผิดพลาดที่โรงงานมักเจอตอน audit
3. **โชว์ของ** (mid funnel): คลิป/ภาพ AI ประเมิน % ความพร้อม + ร่างเอกสารให้
4. **หลักฐาน** (bottom funnel): case study / คำรับรอง (สะสมทันทีที่ได้ลูกค้าแรก)

> สลับเสาให้ครบใน 1 เดือน — อย่าโพสต์แต่ "โชว์ของ" (คนเบื่อ) หรือแต่ "สอน" (ไม่ปิดการขาย)

## 5) 🗓️ Drumbeat รายสัปดาห์ (จังหวะที่ทำได้จริง ~1–1.5 ชม./วัน)
| วัน | งานหลัก (โปรโมชัน) | ผลลัพธ์ที่ต้องได้ |
|---|---|---|
| **จันทร์** | ยิง warm outreach 10 ราย (email/LINE) จากรายชื่อใน tracker | ส่ง 10 |
| **อังคาร** | เผยแพร่คอนเทนต์ 1 ชิ้น (สลับเสา 1–4) + repurpose ลง LINE/FB | โพสต์ 1 |
| **พุธ** | โทร/ตาม 8 คนที่ตอบรับ → นัด concierge | นัด 2–3 |
| **พฤหัส** | ทำ concierge session + โพสต์ LinkedIn (founder 1 ชิ้น) | concierge 1–2 |
| **ศุกร์** | เก็บ social proof/feedback + เข้า FB group ช่วยตอบ 3 โพสต์ + รีวิวตัวเลข | อัปเดต tracker |
| เสาร์-อาทิตย์ | เบา: เตรียมรายชื่อ + ร่างคอนเทนต์สัปดาห์หน้า | คิว 10 lead ใหม่ |

**กฎเหล็ก:** ทำทุกสัปดาห์แม้ผลยังไม่มา — drumbeat ที่สม่ำเสมอ 8 สัปดาห์ > ระเบิดครั้งเดียวแล้วเงียบ

## 6) 📊 วัดผล (North Star + leading indicators)
**North Star:** signup จริง/สัปดาห์ → จำนวนที่จ่ายเงิน
**Leading (ดูทุกศุกร์):**
| ตัวชี้วัด | เป้า/สัปดาห์ |
|---|---|
| Outreach ส่ง | 20 |
| ตอบรับ | 6–8 |
| Concierge นัดได้ | 3–5 |
| ถึง aha (เห็นผล 1 ชิ้น) | 3 |
| **Signup จริง** | **2–3** |
| คอนเทนต์เผยแพร่ | 2 |

**ติด UTM ทุกลิงก์** → `?utm_source=<line/email/linkedin>&utm_medium=outreach&utm_campaign=iso_beachhead`
→ ดูใน GA4 ว่าช่องทางไหนได้ signup จริง + เทียบกับ Supabase (`auth.users` เพิ่มไหม)

## 8) 🔀 เมื่อไรถึงเปลี่ยน segment (Kill criteria)
ทำ drumbeat นี้ครบ **4 สัปดาห์เต็ม** แล้วประเมิน:
- ✅ **ได้ signup ≥ 3 + จ่าย ≥ 1** → ถูกทางแล้ว → **เพิ่มปริมาณ** (scale outreach + จ้างช่วยยิง) ก่อนขยาย segment
- 🟡 ได้ signup แต่ไม่มีใครจ่าย → ปัญหาอยู่ที่ **Process/aha หรือ Price** ไม่ใช่ segment → ปรับ concierge/ราคา ก่อนเปลี่ยน
- 🔴 outreach 80 ราย แต่ตอบรับ < 5% → ปัญหา **message หรือ list** → แก้พาดหัว/เปลี่ยนแหล่งรายชื่อ ก่อนโทษ segment

> อย่าเปลี่ยน segment เพราะ "เบื่อ/ยังไม่มา" ใน 1–2 สัปดาห์ — beachhead ต้องให้เวลาสะสม trust
