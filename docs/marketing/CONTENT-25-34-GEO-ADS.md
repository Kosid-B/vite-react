# คอนเทนต์เจาะ 25-34 + แผนยิงแอด Geo โซนโรงงาน
> อิง GA4 จริง: 25-34 อยู่ 30 นาที/ครั้ง (กลุ่มทอง) · เมืองผู้ใช้ = Rayong, Map Ta Phut, BKK, เชียงใหม่, ขอนแก่น

## ส่วนที่ 1 — คอนเทนต์เจาะกลุ่ม 25-34 (ผู้ประกอบการรุ่นใหม่)

### ทำไมกลุ่มนี้ (จากข้อมูล)
- อยู่นานสุด **29:50/ครั้ง** (35-44 แค่ 2:12) → ผลิตภัณฑ์โดนใจกลุ่มนี้จริง
- โปรไฟล์: เจ้าของธุรกิจ/โรงงานรุ่นใหม่, รับช่วงกิจการครอบครัว, เปิด startup/SME, ถนัดดิจิทัล, หา "ทางลัดที่เชื่อถือได้"

### โทน + มุมคอนเทนต์ (ต่างจากกลุ่มอาวุโส)
| ทำ | เลี่ยง |
|---|---|
| สั้น กระชับ วิดีโอ 9:16 | ย่อหน้ายาว ศัพท์ราชการ |
| "ทางลัด + ประหยัดเวลา/เงิน" | ขายฟีเจอร์ตรง ๆ |
| โชว์ผลลัพธ์จริง (คะแนน %, เอกสารที่ขาด) | ทฤษฎี ISO |
| "รับช่วงโรงงานพ่อแม่ ทำระบบให้ทันสมัย" | น้ำเสียงสั่งสอน |

### 5 คอนเทนต์เจาะ 25-34 (พร้อมทำ)
1. **"รับช่วงโรงงานที่บ้าน แล้วเจอ ISO ค้าง 3 ปี — ผมทำยังไง"** (เล่าเรื่อง relatable) → CEO AI / TIS
2. **"เจ้าของรุ่นใหม่ ไม่ต้องจ้างทีมใหญ่ก็บริหารได้"** — โชว์ทีม AI (CEO/การตลาด/การเงิน) → CEO AI
3. **"เช็กโรงงานพร้อมขอ มอก. กี่ % ใน 5 นาที"** — โชว์ TIS Automate Quick Start → TIS
4. **"ทำไมโรงงานยุคใหม่ต้องมี BCP (แผนสำรองเวลาเกิดวิกฤต)"** → BCMS
5. **"3 เอกสารที่ทำให้โรงงานตก audit บ่อยสุด"** — listicle จี้ pain → bridge

> รูปแบบ: Reels/TikTok/Shorts 9:16 (ใช้เฟรมอินโฟกราฟิก + CapCut ที่ทำไว้แล้ว) · โพสต์เวลาพีค **10 โมง / บ่าย 3** (จาก GA4)

### ช่องทาง
- TikTok/Reels/YT Shorts (กลุ่ม 25-34 อยู่ที่นี่) · bio link → /start, tis-automate, bcms
- Facebook group เจ้าของธุรกิจ/โรงงานรุ่นใหม่
- LINE OA (broadcast คอนเทนต์ + ปิดการขาย)

## ส่วนที่ 2 — แผนยิงแอด Geo โซนโรงงาน

### ทำไม (จากข้อมูล)
GA4 โชว์ผู้ใช้จาก **Rayong + Map Ta Phut** (นิคมอุตสาหกรรมมาบตาพุด/ระยอง = โรงงานหนาแน่นสุดของไทย) + หัวเมืองอุตสาหกรรม (ขอนแก่น, โคราช) → **demand จริงมาจากโซนที่มีโรงงานที่ต้องทำ ISO/มอก./BCMS**

### Geo-target (Facebook/Google Ads)
| โซน | ผลิตภัณฑ์ที่เล็ง | เหตุผล |
|---|---|---|
| **มาบตาพุด / ระยอง** | BCMS + TIS + ISO | นิคมฯ ปิโตรเคมี/หนัก — regulated, ต้อง BCM + ISO |
| **ชลบุรี / อมตะ / แหลมฉบัง** | TIS + ISO + CEO AI | โรงงานผลิต/ส่งออก จำนวนมาก |
| **สมุทรปราการ / บางปู** | TIS + ISO | โรงงาน SME หนาแน่น |
| **ขอนแก่น / โคราช** | CEO AI + TIS | หัวเมืองอุตสาหกรรมอีสาน |
| **ลำพูน (นิคมภาคเหนือ)** | TIS + ISO | อิเล็กทรอนิกส์/ส่งออก |

### โครงแคมเปญ (เริ่มเล็ก ทดสอบก่อน)
- **งบทดสอบ**: ฿300–500/วัน/แคมเปญ (2 สัปดาห์) — เพราะ organic โตดีอยู่แล้ว แอดแค่เร่ง
- **รูปแบบ**: Video 9:16 (คอนเทนต์เจาะ 25-34) + carousel "เช็กโรงงานพร้อมกี่ %"
- **Target**: อายุ 25-45 · ความสนใจ: business owner, manufacturing, ISO, โรงงาน · รัศมีรอบนิคมฯ
- **ปลายทาง**: การ์ด/หน้า landing ตามผลิตภัณฑ์ (มอก.→TIS, BCM→BCMS, ทั่วไป→/start) พร้อม utm

### วัดผล (ก่อนขยายงบ)
| เมตริก | เกณฑ์ผ่าน (ค่อยขยายงบ) |
|---|---|
| CPC | < ฿5 |
| Landing → sign_up | > 10% |
| sign_up → lead/plan | > 5% |
| Cost per lead (TIS/BCMS) | < ฿500 (เทียบดีล ฿50k+ = คุ้มมาก) |

> กติกา: **ยิงเล็ก วัดจริง แล้วค่อยเท** — organic ให้ CAC ต่ำอยู่แล้ว อย่าเผางบก่อนพิสูจน์ conversion (ตาม MARKET-VALIDATION-GA4.md)

## ลำดับลงมือ (2 สัปดาห์แรก)
1. ถ่าย/ทำ 2 คลิปแรก (เจาะ 25-34) จากเฟรม + CapCut ที่มี → โพสต์ organic ก่อน (ฟรี วัด engagement)
2. คลิปไหน engagement ดี → เอาไปเป็น ad creative
3. เปิด geo-ad ฿300/วัน 1 โซน (ระยอง/มาบตาพุด) 2 สัปดาห์ → วัดตามเกณฑ์
4. ผ่านเกณฑ์ → ขยายโซน; ไม่ผ่าน → แก้ creative/landing ก่อนเพิ่มงบ
