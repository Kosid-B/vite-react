ขอ มอก. ต้องทำอะไรบ้าง ปี 2569: ครบทุกขั้นตอน ตั้งแต่เริ่มจนได้ใบอนุญาต

สรุปสำหรับคนไม่มีเวลา: การขอ มอก. (มาตรฐานผลิตภัณฑ์อุตสาหกรรม) คือการขอใบอนุญาตจาก สมอ. (สำนักงานมาตรฐานผลิตภัณฑ์อุตสาหกรรม) เพื่อผลิต/นำเข้า/จำหน่ายสินค้าที่อยู่ในข่ายควบคุม หัวใจอยู่ที่ ระบบควบคุมคุณภาพโรงงาน + เอกสาร + การชักตัวอย่างทดสอบ บทความนี้สรุปครบทุกขั้นตอน จากประสบการณ์ที่ปรึกษา ISO/มอก. กว่า 20 ปี

มอก. คืออะไร ทำไมโรงงานต้องมี

มอก. = มาตรฐานผลิตภัณฑ์อุตสาหกรรม กำกับโดย สมอ. (TISI) ภายใต้กระทรวงอุตสาหกรรม เป็นเครื่องยืนยันว่าสินค้าได้คุณภาพ/ปลอดภัยตามที่มาตรฐานกำหนด แบ่งเป็น 2 ประเภท:

ประเภทความหมาย
มอก. บังคับกฎหมายบังคับ — ต้องได้ใบอนุญาตก่อนผลิต/นำเข้า/จำหน่าย ไม่มี = ผิดกฎหมาย (เช่น เหล็ก ไฟฟ้า วัสดุก่อสร้าง ของเล่น บางรายการ)
มอก. ทั่วไปสมัครใจ — ขอเพื่อเพิ่มความน่าเชื่อถือ/แข่งขันได้ดีขึ้น

ขั้นแรกสุด: ตรวจสอบว่าสินค้าของคุณอยู่ในข่าย มอก.บังคับ หรือไม่ (เช็กรายการควบคุมกับ สมอ. — tisi.go.th)

ขอ มอก. ต้องทำอะไรบ้าง — 6 ขั้นตอนหลัก

  1. ระบุมาตรฐานที่ต้องขอ — เลขที่ มอก. ของสินค้า + ตรวจว่าบังคับหรือทั่วไป
  2. เตรียมโรงงาน + ระบบควบคุมคุณภาพ — สมอ. ประเมิน "ระบบการควบคุมคุณภาพ" ของโรงงาน (มักอิงหลัก ISO 9001) ต้องมีเอกสาร/บันทึกพร้อม
  3. ยื่นคำขอรับใบอนุญาต — ยื่นต่อ สมอ. (ทำผลิตภัณฑ์/นำเข้า) พร้อมเอกสารบริษัท + รายละเอียดผลิตภัณฑ์/กระบวนการผลิต
  4. ตรวจประเมินโรงงาน (Assessment) — เจ้าหน้าที่เข้าตรวจสถานที่ผลิต + ระบบคุณภาพ
  5. ชักตัวอย่างส่งทดสอบ — สุ่มตัวอย่างสินค้าส่งห้องปฏิบัติการทดสอบตามมาตรฐาน
  6. ออกใบอนุญาต + ตรวจติดตามผล — ผ่านแล้วได้ใบอนุญาต + เครื่องหมาย มอก. · หลังจากนั้นมีการตรวจติดตาม (surveillance) เป็นระยะ
จุดที่ SME พลาดบ่อยสุด: ระบบเอกสาร/บันทึกคุณภาพไม่พร้อม — ตรวจโรงงานผ่านยาก เพราะ auditor ขอดู "หลักฐานว่าทำจริง" ไม่ใช่แค่มีเอกสารสวย

เอกสารที่ต้องเตรียม (เช็กลิสต์)

  • เอกสารนิติบุคคล (หนังสือรับรอง + ทะเบียนโรงงาน/ใบอนุญาตประกอบกิจการ)
  • รายละเอียดผลิตภัณฑ์ + แบบ/สเปก + กระบวนการผลิต (flow)
  • คู่มือ/ระเบียบปฏิบัติระบบควบคุมคุณภาพ (Quality Control) + บันทึกการตรวจ
  • บันทึกการควบคุมวัตถุดิบ · การสอบเทียบเครื่องมือวัด · การจัดการของไม่เป็นไปตามข้อกำหนด
  • ผลทดสอบผลิตภัณฑ์ (ถ้ามีจากห้องแล็บ)

ใช้เวลานานแค่ไหน + เตรียมตัวยังไงให้ผ่านเร็ว

ระยะเวลาขึ้นกับความพร้อมของระบบโรงงาน โดยทั่วไป ถ้ามีระบบคุณภาพพื้นฐาน (เช่น ISO 9001) อยู่แล้ว จะเร็วกว่ามาก เพราะ มอก. ประเมิน "ระบบควบคุมคุณภาพ" คล้ายกัน 3 เคล็ดลับ:

  1. ทำ gap analysis ก่อน ว่าขาดเอกสาร/ข้อกำหนดไหน แล้วไล่ปิดทีละข้อ
  2. "เขียนในสิ่งที่ทำ ทำในสิ่งที่เขียน" — อย่าลอกเอกสารบริษัทอื่น
  3. ซ้อมตอบผู้ตรวจ + เตรียมหลักฐาน (บันทึก) ให้ครบก่อนวันตรวจจริง

คำถามที่พบบ่อย (FAQ)

Q: สินค้าของเราต้องขอ มอก.บังคับ ไหม?
A: เช็กรายการสินค้าควบคุมกับ สมอ. (tisi.go.th) ถ้าอยู่ในรายการบังคับ = ต้องขอก่อนผลิต/จำหน่าย

Q: ต้องมี ISO 9001 ก่อนไหม?
A: ไม่บังคับ แต่มีแล้วช่วยมาก เพราะ มอก. ประเมินระบบควบคุมคุณภาพที่ทับซ้อนกับ ISO 9001

Q: ทำเองได้ไหม หรือต้องจ้างที่ปรึกษา?
A: ทำเองได้ถ้ามีคนดูแลระบบคุณภาพ + เอกสารครบ · เครื่องมือช่วยประเมินความพร้อม (เช่น TIS Automate) ช่วยลดเวลาและกันตกข้อกำหนด
