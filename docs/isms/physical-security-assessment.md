## สรุปผลประเมิน (2026-07-07)
สถานที่ทำงานหลัก = **co-working space** (มีการจัดการความปลอดภัยกายภาพระดับมืออาชีพ)

| Control | คำตอบ | สถานะ |
|---|---|---|
| **7.2** Physical entry | เข้าออกด้วย**รหัส (keycode)** · คนภายนอกเข้าถึงที่วางอุปกรณ์ได้ยาก | ✅ ควบคุมแล้ว (จัดการโดย co-working) |
| **7.3** Securing offices | ล็อกห้อง/พื้นที่ได้เมื่อไม่อยู่ · เก็บอุปกรณ์/เอกสารลับใน**ตู้ล็อก** | ✅ |
| **7.4** Physical monitoring | *ยังไม่ยืนยัน CCTV* | 🟡 co-working ส่วนใหญ่มีกล้อง — **แนะนำยืนยันกับผู้ให้บริการ** |
| **7.5** Environmental threats | มีการป้องกันไฟไหม้ · ไม่เสี่ยงน้ำท่วม · ไฟฟ้าเสถียร (โน้ตบุ๊กมีแบตในตัว = backup) | ✅ |
| **7.8** Equipment siting | โน้ตบุ๊กวางในที่ปลอดภัย + เก็บในตู้ล็อก | ✅ |
| **7.13** Equipment maintenance | Windows update · ทำความสะอาด · ตรวจแบต/ฮาร์ดแวร์ | ✅ |

**ข้อเสนอแนะ:** ยืนยัน CCTV กับ co-working (ปิด 7.4) · พิจารณา Kensington lock ล็อกโน้ตบุ๊กกับโต๊ะเมื่อทำงานในพื้นที่ส่วนกลาง

> **บริบท:** ธุรกิจ cloud-first — โครงสร้าง IT การผลิตอยู่บน cloud (physical security ของ data center = inherited จาก provider, อ้าง 5.23/7.1)
> controls หมวดนี้จึงครอบคลุมเฉพาะ**พื้นที่/อุปกรณ์ทำงานจริง**ของผู้ปฏิบัติงาน (โน้ตบุ๊ก + สถานที่ทำงาน)
> IT Security ตรวจระยะไกลไม่ได้ — ต้องเจ้าของธุรกิจสำรวจ+ตอบ
