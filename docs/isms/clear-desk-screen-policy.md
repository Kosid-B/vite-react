## วัตถุประสงค์
ลดความเสี่ยงที่ข้อมูลลับ (credential, source code, ข้อมูลลูกค้า) จะถูกเห็น/เข้าถึงโดยผู้ไม่มีสิทธิ์
เมื่อผู้ปฏิบัติงานไม่อยู่ที่เครื่อง — โดยเฉพาะเมื่อทำงานนอกสถานที่ (ร้านกาแฟ, co-working, สนามบิน)

## นโยบายหน้าจอสะอาด (Clear Screen) — บังคับ
1. **ล็อกหน้าจอทุกครั้งที่ลุกจากเครื่อง** แม้เพียงชั่วครู่ (Windows: `Win + L`)
2. ตั้งให้ Windows **ล็อกอัตโนมัติเมื่อไม่ใช้งานภายใน 5–10 นาที** และ **ต้องกรอกรหัสผ่านเมื่อกลับมา**
   - ตรวจสอบที่ Settings → Accounts → Sign-in options → "If you've been away…" = **When PC wakes up from sleep**
3. ไม่เปิดไฟล์ที่มี credential/ข้อมูลลูกค้าค้างไว้บนหน้าจอเมื่อไม่ได้ใช้งาน
4. ระวังการมองข้ามไหล่ (shoulder surfing) เมื่อทำงานในที่สาธารณะ — พิจารณา privacy screen filter

## นโยบายโต๊ะทำงานสะอาด (Clear Desk) — บังคับ
1. ไม่ทิ้งเอกสารที่มีข้อมูลลับ (รหัสผ่านจดบนกระดาษ, ข้อมูลลูกค้าพิมพ์ออกมา) ไว้บนโต๊ะโดยไม่มีคนดูแล
2. ไม่จดรหัสผ่าน/คีย์ API บน sticky note ติดจอหรือใต้แป้นพิมพ์ — ใช้ password manager แทน
3. เก็บสื่อบันทึกพกพา (USB, external drive) ในที่ล็อกได้เมื่อไม่ใช้ (ถ้ามี)
4. ทำลายเอกสารลับที่ไม่ใช้แล้วด้วยเครื่องย่อยกระดาษ (ไม่ทิ้งถังขยะทั้งใบ)
